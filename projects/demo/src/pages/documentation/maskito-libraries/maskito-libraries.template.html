Maskito is a collection of libraries. Explore them and learn how to install them.

  • @maskito/core

    It is the main zero-dependency and framework-agnostic package. It can be used alone in Vanilla Javascript project. It listens beforeinput and input events to validate and calibrate textfield's value.

    Learn more about this library in "CoreÂ Concepts" section.

  • @maskito/kit

    The optional framework-agnostic package. It contains ready-to-use masks with configurable parameters.

    This package requires @maskito/core as peer-dependency.

    See examples: Number or Time .

  • @maskito/angular

    The Angular-specific library. It provides two convenient ways of using Maskito: basic directive approach (when developer has direct access to native input element) and Dependency Injection approach (when native input element is hidden somewhere deep inside another component).

    This package requires @maskito/core as peer-dependency.

    Learn more about this library in "Angular" section.
