Overwrite mode regulates behaviour of the mask when user inserts new character somewhere in the middle of textfield's value (overwriting the current index's value).

Maskito's options overwriteMode can be 3 different types:

  • shift (default)
  • replace
  • function that returns shift or replace
The most classic mode that everyone is used to. Inserting of a new character in the middle of the textfield's value shifts all right characters at consecutive indexes up. All new inserted characters replace the old characters at the same position. No character shifts. The length of textfield's value remains the same after inserting new character somewhere in middle of the textfield's value. Parameter overwriteMode also accepts function that will called before each insertion of new characters.
