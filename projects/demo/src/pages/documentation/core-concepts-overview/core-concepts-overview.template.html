The main entity of Maskito's core library is Maskito ‑class which accepts 2 arguments:

  1. reference to native HTMLInputElement or HTMLTextAreaElement
  2. set of configurable options MaskitoOptions

To understand the capabilities of the Maskito library, you need to learn about the following features and concepts:

Mask expression

Learn how to predefine your mask format via mask expression. This section describes different types of mask expression and explains meaning of "fixed character"-term.

Processors

Learn about preprocessors and postprocessors, and how to stack them via maskitoPipe .

Overwrite mode

Maskito can behave differently when user inserts new character in the middle of textfield's value. Learn how to control this behaviour via overwriteMode -parameter.
