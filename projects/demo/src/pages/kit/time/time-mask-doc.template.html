Use maskitoTimeOptionsGenerator to create mask for time input.

This generator has the following configurations:

  • mode – time format mode.

    Available options : HH:MM , HH:MM:SS or HH:MM:SS.MSS .

  • timeSegmentMaxValues – max value for every time segment.

    Time segments are units of the time which form time string. For example, HH:MM consists of two time segments: hours and minutes.

HH:MM HH:MM:SS HH:MM:SS.MSS Input time Time format mode Max value for every time segment
