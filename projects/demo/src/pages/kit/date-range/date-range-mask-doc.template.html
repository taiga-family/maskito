Use maskitoDateRangeOptionsGenerator to create a mask to input a range of dates. Use mode and dateSeparator parameters to get a mask with a locale specific representation of dates. Parameters min and max allow you to set the earliest and the latest available dates. They accept native Date . Use minLength and maxLength parameters to set minimal and maximal length of the date range. Use rangeSeparator parameter to customize separator between dates of the date range.
