Use maskitoDateOptionsGenerator to create mask for date input.

This generator has the following configurations:

  • mode – date format mode.

    Available options : DMY , MDY or YMD .

  • max – max value. Accepts native Date

    Default value is null

  • min – min value. Accepts native Date

    Default value is null

  • separator – Symbol for separating date-segments (days, months, years).

    Default value is . (Dot)
