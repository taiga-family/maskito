maskitoWithGuide helps to show placeholder mask characters. The placeholder character represents the fillable spot in the mask.

Creating mask with guide is available only with overwriteMode: 'replace'

This example is the simplest demonstration how to create masked input with guide .

The only required prerequisite is basic understanding of "Mask expression" concept.

The following example shows more complex mask and explains return type of maskitoWithGuide utility.

maskitoWithGuide returns an object which implements MaskitoOptions interface. It contains its own processor and postprocessor , and also defines the only supported overwriteMode: 'replace' .

Also, this complex example uses built-in postprocessor maskitoPrefixPostprocessorGenerator from @maskito/kit .

This last example demonstrates how to integrate maskitoWithGuide with any built-in mask from @maskito/kit .
